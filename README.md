## What this repository is

This repository hosts **Manual 4/20 🌿**, an open manual on 4/20 culture, legal context, coexistence, harm reduction, and responsible organizing.

It is rooted in Bolivia and connected to the ecosystem around [Proyecto Cultural Barranco](https://barranco.life) and the site of [Encuentro Nacional 4/20²⁶ 🌿](https://420.barranco.life), but it is designed to be useful beyond a single event, city, or edition.

## What it is for

The manual exists to help build a more mature, better informed, and more useful public conversation around 4/20 culture, legality, coexistence, and responsible organization.

It also aims to gather, organize, and make accessible useful information for citizens, volunteers, host spaces, journalists, and other people who want to understand the topic without falling into stigma, prejudice, or misinformation.
